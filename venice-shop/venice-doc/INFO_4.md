### venice-vue
本章节主要说明管理后台的前端代码
### 项目包说明
```
venice-vue
|- public 公共页
  |- NEditor  编辑器包
|- src
  |-api 接口
    |-common 部分公共接口
    |-demo demo 页面接口
    |-system demo 页面接口
    |-index
    |-request 请求配置
    |-requestParam 请求参数统一处理／组装
    |-requestUrl 请求地址统一处理／组装
  |-assets 静态资源
  |-components 组件
  |-icons 矢量图标资源
  |-mock 模拟数据生成器 无接口时使用
  |-router 路由
  |-store 状态管理
  |-style 样式配置
  |-utils 部分函数封装
  |-views
    |-common 公共页面
    |-miniapp 商品
    |-system 系统管理

```
